LA LANCHA EMAIL — FONT COMPARISON

Quick Notes:
  • Why not Theinhardt? Theinhardt is a commercial font that must be installed on each user's computer — it won't display in email. Google Fonts (Meie Script + IBM Plex Mono) are free alternatives that look closest to the brand style.
  • Option 1 uses Google Fonts (Meie Script + IBM Plex Mono) — works in ~85% of email clients (Gmail, Apple Mail, webmail)
  • Option 1a shows what Option 1 looks like when fonts are blocked (~15%, mainly Outlook desktop)
  • Option 2 uses 100% universal system fonts (Helvetica + Courier New) — identical on every device, guaranteed
  • All versions have images embedded and can be uploaded directly to RDP
